# Release Title
Warmane Launcher v1.1.4 - Stability & Scalability Update

# Release Description
**Polishing the experience with a fully responsive UI.**

This update addresses reported layout bugs and significantly improves the interface's adaptability. The launcher now scales beautifully across different window sizes, and the sidebar has been enhanced for better accessibility.

# Key Features & Fixes
- **Responsive Design:** The entire interface now scales proportionally with the window size (fluid typography and spacing).
- **Enhanced Sidebar:** Increased sidebar width and button size by 30% for easier navigation.
- **Layout Stability:** Fixed issues where content (like the news section) would shift unexpectedly when the download panel appeared.
- **Improved Positioning:** Adjusted action buttons to sit closer to the main logo for a tighter visual hierarchy.
- **Bug Fixes:** Resolved various UI alignment glitches reported in previous versions.
